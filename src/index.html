Steps to integrate:

  1. This sample page must be run over an HTTP server, not from the file system! The dependent scripts will not load properly if loading from the file system. Likewise with any other integration testbed. HTTP or HTTPS only.
  2. Insert the entire script tag anywhere in your template
  3. Set the width of the cart widget tray as you'd like it to appear. If you omit this, it will default to 470px wide.
  4. NOTE: Each individual site will get unique targetSiteGuid, hostSiteGuid, and path values !! Be sure to set these correctly when we migrate to other environments.
  5. The methods available to you are:
    • __CartWidget.showWidget()
    • __CartWidget.hideWidget()
    • __CartWidget.add()
    • __CartWidget.delete()
    • __CartWidget.goToCart()

    The methods I see you most likely using are add(), goToCart() and showWidget(). hideWidget() and delete() are taken care of in my script functionality.
